# Zymmy Club Gym Website

A premium, interactive gym website built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

## Features

- **Interactive Gallery**: Smooth, auto-scrolling image carousel.
- **Video Section**: World-class facilities section with a YouTube video modal.
- **Classes Timetable**: Hover-responsive class schedule.
- **Testimonial Slider**: Premium slide-in client reviews.
- **Blog Section**: News grid with hover-revealed background images.
- **Logo Slider**: Automatically scrolling client logo track.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone or download the project files.
2. Open your terminal in the project directory.
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server and view the website:
```bash
npm run dev
```
The site will be available at `http://localhost:5173`.

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The built files will be in the `dist` directory.

## Technologies Used

- **React**: Core UI library.
- **Vite**: Modern build tool and dev server.
- **Tailwind CSS**: Rapid UI styling and layout.
- **Framer Motion**: High-performance animations and transitions.
- **Font Awesome**: Premium icon set.
